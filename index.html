Paul's Planner

Paul's Planner is a full-stack Rails Application, the purpose of the app is to help facilitate effective collaboration between teachers and students. It uses HTML, CSS, Javascript, Bootstrap, and SASS for the front-end and PostgreSQL for the database. The Omniauth and devise gems are used for user authentification. Cancancan was used for authorization. It also integrates the Google Maps API for meetings. Tests were written with RSpec/Capybara.

Paul's Planner was a group project at LEARN Academy where we followed Agile methodologies and our group wrote our own user stories.
